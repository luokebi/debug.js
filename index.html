debug.js

Features:

Ability to set prefix for the content of what you want to log.

debug.setPrefix("XXX:");
debug.log("hello world"); // log "XXX:hello world"

Ability to disable all method.

debug.disable();
debug.log("hello world"); // nothing will to log
